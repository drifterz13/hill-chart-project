# Hill chart progress tracker

This is a simple progress tracker that uses a hill chart to visualize the progress of tasks. It is built using Bun fullstack dev server.


![App screenshot 1](./assets/todoi-1.png)
![App screenshot 2](./assets/todoi-2.png)


To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.1. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
